# DinosauriAR
Anleitung für die Verwendung des Projekts „DinosauriAR“

Die Anwendung wurde mit Unity 2018.4.2f1 (Long Term Support: https://unity3d.com/unity/qa/lts-releases) inklusive Android Build Support und Android SDK 7.0 entwickelt und getestet. Bei Nutzung von anderen Versionen kann die Funktionstüchtigkeit nicht gewährleistet werden. 

Im Ordner "Builds" existiert ein vollständiger Build in einer apk-Datei ("DinosauriAR.apk").
Für die Ausführung auf einem Smartphone muss in den Einstellungen aktiviert werden, dass unbekannte Apps installiert werden dürfen. Außerdem werden die „Google Play-Dienste für AR“ benötigt, welche jedoch automatisch während der Anwendung installiert werden können (ein Google-Account wird benötigt). Des Weiteren muss der Zugriff auf die Kamera erlaubt sein und eine Verbindung mit dem Internet bestehen. 
Die App sollte auf jedem ARCore-fähigen Smartphone funktionieren. Getestet wurde auf einem Google Pixel 2, einem Samsung Galaxy S8+ und einem Samsung Galaxy S8. 

Zur besseren Verdeutlichung wurden 2 unterschiedliche Demos aufgezeichnet. Diese sind im Ordner „Demos“ zu finden und zeigen Ausschnitte der Tendaguru-Expedition aus der Sicht des Hosts und eines Clients. 

Die benötigten Werkzeuge können mithilfe der Dateien im Ordner „Werkzeuge“ ausgedruckt und mithilfe der Demos zusammengebaut werden.
